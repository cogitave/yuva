## Summary

Forward task backlog for TABOS (from-scratch no_std Rust framekernel) synthesized from mined deferred-hooks, the L2 sovereignty chain, verification gaps, portability bugs, perf/bench, and docs debt. The v1 (M0-M4) + v2 agent-native chain (M5-M18) are complete and CI-green (TCG + tb-vmm/KVM); the L2 microhypervisor track has started at L2.0 (VMX-root, builds + boots via a graceful-skip path that proves nothing about the world switch). Ordering is leverage-first: (1) the F2 M8 LAPIC/timer fault under QEMU-microvm+KVM is a real boot bug that also unblocks the KVM bench full number; (2) the F1 nested-VMX validation substrate gates every L2.0-L2.6 real hardware-virt proof since hosted CI has no nested VMX; (3) the M11 rights-subset/confused-deputy formal proof is load-bearing for M18's entire frozen-boundary safety guarantee. These three are P0. Then Miri-in-CI (mandatory Tier-0) and the L2.0 real VMLAUNCH proof, then the L2.1..L2.9 chain in strict dependency order (each its own task), the aarch64 tb-boot/EL2 parallel track (CI-friendly TCG world-switch proof), then the deferred features (byte-payload IPC, blocking-recv IPC, durable VirtioBlkStore, real inference adapters + virtio transport, M18 human-approval gate, M18 held-out evaluator, M15 unmap/reclaim), broader Kani harnesses, and finally perf-bench + docs reconciliation. Grounded against crates/tb-hal/src/arch/x86_64/vmx/ (controls/ept/exit/vmcs/vmxon/world_switch.rs), timer.rs (LAPIC at PML4[3] VA 0x0000_0180_0000_0000 -> PA 0xFEE0_0000), caps.rs (args[2..3] reserved buffer + rights-subset invariant doc), ipc.rs (copy_to/from_user deferred + WouldBlock), mem.rs (BackingStore/RamStore, VirtioBlk deferred), scripts/bench-boot.sh (FINAL_MARKER='L2.0: vmxroot OK', -M microvm full=NA), and .github/workflows/{ci,vmm-boot,l2-nested-vmx}.yml. 30 tasks.
